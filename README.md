# МедВызов: регистрация заявок вызова СМП (клиентская часть) 

Демо‑версия прототипа медицинской информационной системы.  
Клиентская часть MedCall.Registration.Client Desktop‑приложение (C# + Avalonia + ReactiveUI MVVM + DI).

## Описание проекта

«МедВызов» — демонстрационная версия клиентского приложения для регистрации заявок на вызов скорой медицинской помощи (СМП).  Проект демонстрирует:
* архитектуру приложения с разделением на слои MVVM и применение Reactive;
* интеграцию с REST API серверной части;
* возможности использования библиотеки Avalonia для построения сложного пользовательского интерфейса;
* подходы к структурированию логики, использованию DI для обеспечения слабой связанности.

## Функциональные возможности

**Раздел «Заявки»:**
* отображение списка заявок в гриде с детализацией информации по выбранной заявке;
* фильтрация заявок по статусам;
* подсчёт количества заявок по фильтрам статусов;

**Раздел «Бригады»:**
* визуализация информации о бригадах СМП в виде грида;
* отображение загрузки выбранной бригады в виде тайм-лайн диаграммы (to-do)
* фильтрация и подсчёт их количества бригад по статусам;

## Технологии и стек

* **Платформа:** Win32 .Net10;
* **Язык программирования:** C#;
* **Компоненты:** Avlonia, ReactiveUI MVVM, DI;
* **Формат данных:** JSON;
* **Интерфейс взаимодействия:** REST API (интеграция с серверной частью `МедВызов`).

## Запуск и настройка

### Предварительные требования

* Среда разработки **Visual Studio** с установленным расширением **Avalonia**;
* запущенный сервер `МедВызов` (доступен по адресу `http://localhost:5285`).

### Локальный запуск

1. После сборки поместите файлы в соответствующие каталоги.
2. Запустите исполняемый файл сервера тестовых данных.    
3. Запустите исполняемый файл приложения.
